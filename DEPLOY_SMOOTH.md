## 🎯 Deployment Steps:

### Step 1: Verify Changes
```bash
# See what's changed
git status
git diff app/globals.css
```

### Step 2: Stage Files
```bash
# Add all changes
git add .
```

### Step 3: Commit
```bash
# Make a commit
git commit -m "🎬 Add smooth animations to entire portfolio

- Added 150+ lines of smooth animations
- Created smooth-scroll utility
- Created page-transition components
- Added animations guide documentation
- Performance optimized for 60 FPS"
```

### Step 4: Push
```bash
# Push to main (or seo-friendly-portfolio)
git push origin main

# OR push to feature branch first:
git push origin seo-friendly-portfolio
```

### Step 5: Verify Deployment
```
1. Wait 2-3 minutes
2. Visit: https://pratiksharma.vercel.app
3. Refresh page (Ctrl+R or Cmd+R)
4. Test animations on desktop and mobile
```

---

## 🔄 If Something Goes Wrong:

### Rollback (Undo the changes):
```bash
# See commit history
git log --oneline

# Revert to previous commit
git revert HEAD
git push origin main

# OR reset completely
git reset --hard HEAD~1
git push -f origin main
```
